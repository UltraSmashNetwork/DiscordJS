# Discord.JS Bot

Fully Customizable Basic Discord.JS Bot

## Usage

config.js
```
exports.Prefix = `PREFIX`;//your prefix for bot
exports.Token = `TOKEN`;//your token 
exports.Color = `COLOR`;//color   of embed
```
